Tutorial 1: Bash Basics - Navigation & Files

What is Bash?

Bash is the command-line interface (shell) for Linux. Instead of clicking icons, you type commands to interact with your computer.

Lesson 1.1: Where Am I? (pwd)

Command: pwd (Print Working Directory)

bash
pwd

What it does: Shows your current location in the file system.

Try it now! Open Ubuntu and type pwd, then press Enter.

Expected output: Something like /home/yourusername

Lesson 1.2: What's Here? (ls)

Command: ls (List)

bash
ls

What it does: Lists all files and folders in your current directory.

Try these variations:

bash
ls          # Basic list
ls -l       # Long format (shows details)
ls -la      # Long format including hidden files (files starting with .)
ls -lh      # Long format with human-readable file sizes

Practice:

  1. Run ls
  2. Run ls -l
  3. Compare the outputs

Lesson 1.3: Moving Around (cd)

Command: cd (Change Directory)

bash
cd directory_name    # Go into a directory
cd ..               # Go up one level (to parent directory)
cd ~                # Go to your home directory
cd /                # Go to root directory
cd -                # Go back to previous directory

Practice Exercise:

bash
pwd                 # See where you are
cd /                # Go to root
pwd                 # See you're at /
ls                  # See what's in root
cd ~                # Go back home
pwd                 # See you're home

Important Concept:

  • . means "current directory"
  • .. means "parent directory"
  • ~ means "your home directory"
  • / means "root directory" (top of file system)

Lesson 1.4: Creating Directories (mkdir)

Command: mkdir (Make Directory)

bash
mkdir folder_name           # Create one folder
mkdir folder1 folder2       # Create multiple folders
mkdir -p path/to/folder     # Create nested folders

Practice Exercise:

bash
cd ~                        # Go home
mkdir test_folder           # Create a folder
ls                          # See it in the list
cd test_folder              # Go into it
pwd                         # See where you are

Lesson 1.5: Creating Files (touch)

Command: touch (Create empty file)

bash
touch filename.txt          # Create empty file
touch file1.txt file2.txt   # Create multiple files

Practice Exercise:

bash
cd ~/test_folder            # Make sure you're in test_folder
touch myfile.txt            # Create a file
ls                          # See the file
ls -l                       # See details about the file

Lesson 1.6: Viewing Files (cat, less)

Command: cat (Concatenate and display)

bash
cat filename.txt            # Display entire file

Command: less (View large files page by page)

bash
less filename.txt           # View file (press 'q' to quit)

Practice Exercise:

bash
cd ~/test_folder
echo "Hello World" > myfile.txt    # Put text in file
cat myfile.txt                      # View the file

Lesson 1.7: Copying Files (cp)

Command: cp (Copy)

bash
cp source.txt destination.txt       # Copy file
cp source.txt /path/to/destination/ # Copy to another location
cp -r folder_name new_folder        # Copy entire folder (-r = recursive)

Practice Exercise:

bash
cd ~/test_folder
cp myfile.txt myfile_copy.txt       # Make a copy
ls                                   # See both files

Lesson 1.8: Moving/Renaming Files (mv)

Command: mv (Move)

bash
mv oldname.txt newname.txt          # Rename file
mv file.txt /path/to/destination/   # Move file

Practice Exercise:

bash
cd ~/test_folder
mv myfile_copy.txt renamed.txt      # Rename the copy
ls                                   # See the new name

Lesson 1.9: Deleting Files (rm)

Command: rm (Remove) - CAREFUL! No undo!

bash
rm filename.txt                     # Delete file
rm -r folder_name                   # Delete folder and contents
rm -rf folder_name                  # Force delete (BE VERY CAREFUL!)

Practice Exercise:

bash
cd ~/test_folder
rm renamed.txt                      # Delete the renamed file
ls                                  # See it's gone

Lesson 1.10: Getting Help

Command: man (Manual)

bash
man ls          # Read the manual for ls command
man cp          # Read the manual for cp command

Press 'q' to quit the manual

Also useful:

bash
ls --help       # Quick help for most commands
